# 📊 Project Management System

## 📌 Overview

This is a simple **Project Management System** built using **HTML, CSS, and JavaScript**.
It allows users to create projects, add tasks, and manage them efficiently.

---

## 🚀 Features

* 🔐 User Login (Frontend-based)
* 📁 Create and delete projects
* ✅ Add multiple tasks under each project
* 📝 Task details:

  * Title
  * Description
  * Status (Pending / In Progress / Completed)
  * Priority (Low / Medium / High)
  * Due Date
* 🔄 Update task status
* ❌ Delete tasks and projects
* 🔍 Search tasks
* 🎯 Filter tasks by status
* 📱 Responsive UI (Mobile + Desktop)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* LocalStorage (for data storage)

---

project/

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Login with any username and password
4. Start creating projects and tasks

---

## 🔐 Login Info

* Any username and password will work
* Data is stored locally using browser storage
